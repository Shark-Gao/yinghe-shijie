# 硬核视界

「硬核视界」是一个面向男性用户的海外优质视频精选项目，重点沉淀账号定位、选题库、视频推荐、制作流程和发布记录。

## 账号定位

面向喜欢科技、AI、军事、商业、汽车机械、纪录片、心理学、男性健康与关系科学的男性观众。

核心方向：

- 视觉冲击强
- 信息密度高
- 少真人出镜或无真人出镜
- 适合中文二创解说
- 适合在 Bilibili / 抖音 / 视频号等平台持续更新

## 内容优先级

优先选择 YouTube 上画面质量高、靠视觉讲故事的视频：

- 3D 动画科普
- 工程机械拆解
- 军事地图动画
- 工业制造过程
- AI 演示
- 大型机械与基础设施
- 纪录片素材
- 地图与数据可视化

## 内容矩阵

| 类型 | 建议比例 |
|---|---:|
| 科技纪录片 | 30% |
| AI | 20% |
| 军事 | 15% |
| 世界冷知识 | 15% |
| 汽车机械 | 10% |
| 人体科普 | 10% |

## 追踪原则

- 可以重复推荐同一个 YouTube 频道。
- 不要重复推荐同一个源视频。
- 优先做主题系列，而不是每天大幅切换内容方向。
- 每次推荐、制作、发布后，及时更新对应记录文件。
