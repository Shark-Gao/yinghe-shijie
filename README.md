# 硬核视界

「硬核视界」是一个面向中文受众的硬核知识短视频内容项目，以优质海外视频为视觉素材，面向抖音、快手、小红书、B 站等平台制作并分发适配版本。

## 账号定位

面向喜欢科技、AI、军事、商业、汽车机械、纪录片、心理学、男性健康与关系科学的男性观众。

核心方向：

- 视觉冲击强
- 信息密度高
- 少真人出镜或无真人出镜
- 适合中文二创解说
- 适合在抖音 / 快手 / 小红书 / Bilibili 等平台持续更新
- 一源多剪、多平台分发：按平台调整开头、节奏、时长、标题与封面

## 内容优先级

优先选择 YouTube 上画面质量高、靠视觉讲故事的视频：

- 3D 动画科普
- 工程机械拆解
- 军事地图动画
- 工业制造过程
- AI 演示
- 大型机械与基础设施
- 纪录片素材
- 地图与数据可视化

## 国内化选题思路

参考国内科技、工程、AI、军事和地理科普类 UP 主的常见选题，推荐外网源视频时优先判断它能不能转化成国内观众更熟悉的表达：

- 大国工程：核电站、水电站、高铁、桥梁、隧道、电网、港口、航天工程。
- 产业竞争：芯片、光刻机、GPU、机器人、电池、发动机、无人机、机床。
- AI 应用：AI Agent、AI 训练、AI 硬件、AI 视频、AI 安全、AI 机器人。
- 军工装备：航母、潜艇、坦克、战斗机、导弹防御、无人作战系统。
- 世界冷知识：地图动画、国家地理、极端环境、资源争夺、灾难复盘、超级建筑。

选题不要停留在“某技术如何工作”，要尽量包装成一个国内观众能立刻理解的问题或冲突，例如：

- `芯片为什么这么难造？`
- `AI 为什么离不开 GPU？`
- `航母内部到底怎么运转？`
- `中国最想突破的机器是什么原理？`
- `这个超级工程为什么只有少数国家能做？`

## 追踪原则

- 可以重复推荐同一个 YouTube 频道。
- 不要重复推荐同一个源视频。
- 优先做主题系列，而不是每天大幅切换内容方向。
- 每次推荐、制作、发布后，及时更新对应记录文件。
